# Student Course Management System

## Description
This is a web-based Student Course Management System developed using Java Servlets, JDBC, MySQL, HTML, and CSS.

## Features
- Student Registration
- Student Login
- Session Management
- View Courses
- Enroll Courses
- Profile Management
- Logout

## Technologies Used
- Java
- Servlets
- JDBC
- MySQL
- HTML
- CSS
- Apache Tomcat

## Server
Apache Tomcat 9

## Database
MySQL
